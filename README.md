# Surrey Market Analysis Dashboard

A real estate market analysis dashboard for Surrey, focusing on condo market trends and pricing strategies.

## Features

- Construction Activity Analysis (Starts and Completions)
- Historical Performance Metrics
- Competitive Market Analysis
- Macro Factor Analysis
- Price Optimization Strategy

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the dashboard:
```bash
streamlit run src/dashboard.py
```

## Data Sources

- Construction data from CMHC
- Market pricing data from MLS
- Macro indicators from Statistics Canada

## Deployment

This dashboard is deployed on Streamlit Community Cloud.
